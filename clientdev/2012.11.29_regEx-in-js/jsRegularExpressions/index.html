Meta Characters

  • . - (Period) Matches any character
  • * - (Asterisk) Matches the previous element (character or group) zero or more times
  • + - (Plus) Matches the previous element (character or group) one or more times
  • ? - (Question Mark) Matches the previous element zero or one time
  • ? - (Question Mark) Modifies the previous Meta Character to not be greedy
  • \ - (Back Slash) Escapes the next character so the parse treats it special
  • | - (Vertical Bar) Matches either the preceding or the next element
  • - - (Hyphen) When used in a square bracket group, matches ranges

Meta Characters - Part 2

  • ^ - (Carrot) Matches the beginning of the string (or line)
  • $ - (Dollar Sign) Matches the end of the string (or line)

Grouping

  • (abc) - (Parenthesis) Capturing
  • (?:abc) - (Parenthesis w/ Question Colon) Non-capturing
  • [abc] - (Square Brackets) "One of these"
  • [^abc] - (Square Brackets w/ Carrot) "None of these"

Escape Sequences

  • \b - Word Boundary
  • \w - Word Character [A-Za-z0-9_]
  • \W - Non-Word Character [^A-Za-z0-9_]
  • \s - Whitespace Character (space, tab, line feed, form feed, carriage return...)
  • \S - Non-Whitespace Character (all the rest)
  • \d - Numeric Digit [0-9]
  • \D - Non-Numeric Digit [^0-9]
